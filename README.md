## 배경
NEWYES 씽크펜이란 제품을 우현히 발견. 오프라인 공책에 그림이 전자상으로 바로 표현되는 것에 흥미. 태블릿 없이 전자상으로 표현 가능, 종이책에 필사한 내용이 전자상으로 표현되는 것에 의의.
하지만 가격이 비쌈 + 전용노트가 필요함
원리를 파악해보니 전용 노트에 안 보이는 식별 문양을 그리고 카메라나 센서로 확인함.

전용노트를 사용하는 불편함 -> 일반적인 노트에서도 활용 가능하면 좋을 듯

## 목적
IMU 센서를 활용하여 범용 노트에서도 활용가능한 씽크펜 제작.

### 개인적인 목표
임베디드에 대한 기본적 이해 
STM의 HAL 함수 기본으로 사용하여 개발
추후 PCB 아트웍 및 상용화 과정 체험
특허 제출 경험

## 개발과정

### PART 1 : Core Function

씽크펜 기초 하드웨어 제작
위치 특정 및 궤적 구하기
씽크펜 앱을 통해 궤적 표시


### PART 2 : Correction Algo

위치 보정 알고리즘
궤적 보정 알고리즘

### PART 3 : Core UI/UX

씽크펜 앱의 UI/UX 개발


### PART 4 : Commercialization

PCB 개발
HW / SW 상용화
특허 출원

### PART 5 : Addtional Function

햅틱, LED 기능 등 추가
